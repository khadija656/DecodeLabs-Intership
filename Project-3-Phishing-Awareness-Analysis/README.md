# Phishing Awareness Analysis

## Project Overview

This project was completed as **Project 3** of the **DecodeLabs Industrial Training Program 2026** under the Cyber Security track.

The objective of this project is to analyze phishing emails, identify common phishing indicators, and understand how attackers use social engineering techniques to trick users into revealing sensitive information. Five simulated phishing emails representing well-known organizations were created and analyzed to demonstrate real-world phishing scenarios.

---

## Objectives

* Understand the concept of phishing attacks.
* Identify suspicious sender addresses and malicious URLs.
* Detect common phishing indicators and social engineering techniques.
* Evaluate the security risks associated with phishing emails.
* Improve cybersecurity awareness and threat analysis skills.

---

## Simulated Organizations

The following phishing scenarios were analyzed:

* Netflix
* HBL Bank
* Instagram
* Amazon
* Google

---

## Analysis Performed

Each phishing email was examined for:

* Fake sender email address
* Suspicious hyperlinks
* Urgent or threatening language
* Requests for sensitive information
* Social engineering techniques
* Risk level assessment
* Recommended security practices

---

## Key Security Recommendations

* Verify the sender's email address before responding.
* Never click suspicious or shortened links.
* Access websites using their official URLs.
* Enable Multi-Factor Authentication (MFA).
* Never share passwords or OTPs through email.
* Report phishing emails immediately.
